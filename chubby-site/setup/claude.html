Setup Guide

Connect to Claude

Run Monte Carlo simulations in Claude Desktop or claude.ai. Fragment URLs mean your data never leaves your browser.

Requires Claude Pro/Max/Team/Enterprise with Connectors enabled.

Claude Desktop

1

Open Claude Desktop

Launch the Claude Desktop application.

2

Add Connector

Go to Settings → Connectors → Add connector

3

Enter the MCP Server URL

https://api.chubby.fyi/mcp

Authentication: None (no API key required)

4

Save and Chat

The connector is available in your next conversation.

Claude.ai (Browser)

1

Go to claude.ai

Sign in at claude.ai

2

Add Connector

Settings → Connectors → Add connector

3

Paste URL

https://api.chubby.fyi/mcp
4

Enable and Start

Toggle the connector on and start chatting.

Mobile tip: add the connector on web/desktop first, then use it on mobile.

Test Your Connection

Try this prompt to verify the connector is working:

Run a simulation: age 55, $400k saved, no income, spending $80k/year. Seed 22222.
Run a simulation: age 55, $400k saved, no income, spending $80k/year. Seed 22222.

Claude should invoke the run_simulation_packet tool and return a fragment URL pointing to widget.chubby.fyi.

Privacy Advantage

Claude's connector returns fragment URLs. The #fragment portion of a URL is never sent to servers — this is browser security, not a choice we make.

https://widget.chubby.fyi/#d=eJxLTEo... #fragment never reaches servers

This means your simulation data exists only in your browser. We can't see it, store it, or share it.
